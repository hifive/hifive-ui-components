Class: MagnetController

MagnetController

new MagnetController()

要素のトラック移動及び、要素同士が近づいたら磁石のようにくっつくような挙動をさせるコンポーネント

このコントローラのバインド対象要素の子要素をマグネット要素として扱い、マグネット要素をトラック移動可能にします。さらに、マグネット要素同士が近づいた時に磁石のようにくっつく挙動をするようになります。

使用方法はマグネットコンテナを参照してください。

動作サンプルはこちら

このコントローラをバインドすると、マグネット要素操作時に以下のイベントがあがるようになります

magnet
マグネット要素同士がくっついた時に上げるイベント。
トラック移動したマグネット要素ではなく、トラック移動した要素がくっついた要素から上がるイベントです。
magnetイベント引数には以下のようなオブジェクトが渡されます。

{
	$target: $target, // 移動させた要素(jQueryオブジェクト)
	$group: $group, // くっついた要素を含む、その要素とつながっている要素(jQueryオブジェクト)
	MAGNET_PLACE: MAGNET_PLACE // 要素のくっついた位置を表す定数
	place: place, // 移動させた要素がくっついた要素から見て、移動した要素がくっついた位置を表したMAGNET_PLACEに定義されている数値
}
MAGNET_PLACEは以下のような定数を保持しています。
プロパティ名 値 意味
NONE 0 どこにもくっついていない状態
TOP 1 上部にくっついた
RIGHT 2 右にくっついた
BOTTOM 3 下にくっついた
LEFT 4 左にくっついた
PILED 5 重なった
unmagnet
くっついているコンテナがトラック移動で剥がされた時に、トラック移動したマグネット要素からunmagnetイベントが上がります。
イベント引数はありません。
piled
マグネット要素を移動して、他のマグネット要素と重なった時に上がるイベントです。
トラック移動したマグネット要素から上がります。
イベント引数は以下のようなオブジェクトです。

{
	$target: $target, // 移動させた要素(jQueryオブジェクト)
	$piled: $piled, // 移動させた要素と重なった要素(jQueryオブジェクト)
}
h5mag-trackstart
マグネット要素のトラック移動が開始された時に、その要素からh5mag-trackstartイベントが上がります。
イベント引数には、元のh5trackstartイベントオブジェクトが渡されます
h5mag-trackmove
マグネット要素のトラック移動時に、その要素からh5mag-trackmoveイベントが上がります。
イベント引数には、元のh5trackmoveイベントオブジェクトが渡されます
h5mag-trackend
マグネット要素のトラック移動終了時に、その要素からh5mag-trackendイベントが上がります。
イベント引数には、元のh5trackendイベントオブジェクトが渡されます

Members

(static) config :Object

マグネットコンテナの設定オブジェクト

以下のプロパティが設定可能です。

ignorePattern
string|DOM|jQuery
マグネットコンテナの制御対象外にする要素。セレクタもDOM要素も指定可能です。jQuery.is()で判定します。デフォルトnull。
replaceMode
boolean
コンテナ同士がくっついている箇所に別のコンテナを重なるように移動した時、重なったコンテナ同士の位置の入れ替えを行うかどうか。デフォルトはfalseです。
pileMode
boolean
コンテナ同士がくっついている箇所に別のコンテナを重なるように移動した時、移動したコンテナを同じ個所にくっつけるかどうか。デフォルトはtrueです。
pipeModeをtrueに設定した場合は、replaceModeの設定は無視されて、pipeModeの設定が優先されます。
