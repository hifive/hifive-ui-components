hifiveVisualEditor

hifiveを使用したwysiwygエディタです。

デモサイト
/hifiveVisualEditor/editor
詳細説明
ブラウザ上で動作するwysiwygエディタです。
主な機能
  • ブラウザ上でのHTML作成
  • 既存のページの編集機能
  • 編集しながら他の端末(タブレット等)に反映される自動更新機能
  • CSSライブエディタ
